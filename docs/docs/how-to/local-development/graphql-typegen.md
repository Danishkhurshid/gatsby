## Introduction

If you're already using [Gatsby with TypeScript](/docs/how-to/custom-configuration/typescript) and manually typing the results of your queries, you'll learn in this guide how Gatsby's automatic GraphQL Typegen feature can make your life easier. You don't need to type your queries by hand anymore as you can rely on the types that are generated by Gatsby itself.
